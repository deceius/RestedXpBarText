## RestedXpText | Deceius

### Overview
Show Rested XP percentage on Exp Bar.

### Settings
- `/restedxp` - show help
- `/restedxp percentage` - show percentage
- `/restedxp remaining` - show remaining xp
- `/restedxp both` - show both setting
